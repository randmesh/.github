__Randmesh is an open ecosystem for verifiable and composable randomness systems__. It provides a trust-minimized framework for generating, combining, and verifying entropy from multiple sources in a deterministic and auditable way.

The project is structured around a Go-based backend that implements the core cryptographic primitives and API layer. This includes a reference implementation of provably fair randomness generation, commit-reveal workflows, and integration with external entropy sources such as blockchain-based beacons and distributed randomness networks.

Randmesh is designed for systems where randomness must be provable, reproducible, and resistant to manipulation, including gaming systems, distributed protocols, and audit-critical applications.
