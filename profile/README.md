__Randmesh is an open ecosystem for verifiable and composable randomness systems__. It provides a trust-minimized framework for generating, combining, and verifying entropy from multiple sources in a deterministic and auditable way.

The project is structured around a Go-based backend that implements the core cryptographic primitives and API layer. This includes a reference implementation of provably fair randomness generation, commit-reveal workflows, and integration with external entropy sources such as blockchain-based beacons and distributed randomness networks.

Randmesh is designed for systems where randomness must be provable, reproducible, and resistant to manipulation, including gaming systems, distributed protocols, and audit-critical applications.

Randmesh can be used for a wide range of cryptographic and application-level randomness tasks, including secure random number generation, private key and seed creation, OTP/SMS/email verification codes, session tokens, UUIDs, API secrets, salts, nonces, probability rolls (0–100 and arbitrary ranges for gaming or casino-style systems), randomized identifiers, and generation of random text strings up to 256 characters or more. The platform is designed for systems where every generated value must be unique, unpredictable, verifiable, and resistant to manipulation.

Randmesh does not assume trust in any single entropy source. Randomness is derived from composable multi-source entropy mixing, deterministic verification workflows, and publicly auditable commit-reveal pipelines. Even if one or more sources are biased or compromised, the final output remains unpredictable as long as at least one entropy contributor remains honest.

Unlike many randomness APIs that rely on infrequent beacon updates (for example once per minute) or impose strict free-tier limits, Randmesh generates fresh entropy for every request in real time. There are no artificial monthly caps, paid unlocks after a small number of requests, or recycled outputs between calls. Each API invocation produces a newly derived random value based on composable entropy sources and deterministic verification workflows, while remaining fully free and publicly accessible.
